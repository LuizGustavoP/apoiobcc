25 anos depois da "chegada" do Linux na USP[1], temos o prazer de apresentar o grupo FLUSP -- FLOSS (Free/Libre/Open Source Software) at USP.

Estamos formalizando o FLUSP como grupo de extensão no IME-USP, com o objetivo de estudar e contribuir para o desenvolvimento de projetos de software livre. Dentre outros projetos, temos membros atualmente contribuindo para o Kernel Linux e o compilador GCC, inclusive via disciplinas do MAC.

E porque contribuir com projetos de software livres?

  • Aprender com códigos de sucesso
  • Aplicar para programas [bem remunerados] como o Google Summer of Code e Outreachy.
  • Interagir com grandes comunidades de software.
  • Visibilidade (já ganhamos 3 hardwares via doações de uma empresa).
  • Oportunidades de boas posições em grandes "players" do software livre.
  • Aprender a revisar códigos escritos por outros.
  • Por "a mão na massa" em códigos de grandes projetos de software de diversas áreas, como Linux, tensorflow, love2d, etc.
  • Ganhar experiência via o código daquele sistema que você sempre pensou "Caraca, como isso é feito?"
  • Você terá a oportunidade de interagir com desenvolvedores de alto nível espalhados ao redor do mundo.

Convidamos todos para participarem e contribuírem para o kernel, gcc e/ou outros projetos :) Se você já escreve código para algum projeto de software livre, também é muito bem vindo para fazer parte do grupo e compartilhar os seus conhecimento com outros colegas via FLUSP. Auxiliar uns aos outros para ingressar nas comunidades é justamente a ideia do grupo!

A partir de 2019, nos reuniremos todas as sextas-feiras, das 14h às 18h (local a definir), mas você já pode ler um pouco mais sobre o grupo em flusp.ime.usp.br/about. Se você deseja participar, basta entrar em contato através dos meios abaixo :)

  • 1 - através da nossa lista de emails flusp@googlegroups.com
  • 2 - ou em nosso grupo do telegram (https://t.me/joinchat/BDEctw1URj59ayCWY88qvA).

Lembramos, também, que você pode aproveitar seu empenho no FLUSP e ainda ganhar créditos com isso, se matriculando nas seguintes matérias:

  • MAC0214 - Atividade Curricular em Cultura e Extensão
  • MAC0215 - Atividade Curricular em Pesquisa

Ou propor que o projeto que você colabora no FLUSP seja alvo em matérias como:

  • MAC0342 - Laboratório de Programação eXtrema
  • MAC0470 - Desenvolvimento de Software Livre

Ou, ainda, fazer seu Trabalho de Conclusão de Curso ou uma Iniciação Científica baseado em um software livre, com apoio do FLUSP.

Quaisquer duvidas, não hesitem em perguntar! Esperamos vocês!
